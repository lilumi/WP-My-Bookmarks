== Description ==

Plugin allows you to add posts to your bookmarks after user’s login. 
Have a button and icon “Add to Bookmarks” for each post in the archives, clicking on that button post will be saved. 
In the menu, there is a link to the page “Bookmarks” with a list of added posts (picture, title, short description). 
You can delete posts from the list.

== Installation ==

1. Upload `wp-my-bookmarks.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can add shorcode `[bookmarks]` on a page to get list of user’s bookmarks.
